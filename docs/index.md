# Introducción a la Tarea

Bienvenido a la documentación de esta actividad. El objetivo principal de este proyecto es registrar y explicar todos los pasos técnicos involucrados en la configuración de un entorno de desarrollo y despliegue completo.

Este sitio web, construido con **MkDocs**, documenta el proceso completo, abarcando desde la creación inicial del repositorio de código fuente hasta su publicación final como un sitio estático servido a través de un contenedor.

---

## Índice de Contenidos

Aquí puedes navegar a las diferentes secciones que componen la documentación de la tarea:

* **[Proceso Git](git.md)**
  * Se detalla el proceso y las operaciones realizadas para la creación y configuración del repositorio en Git.

* **[Configuración de GitHub Actions](gitActions.md)**
  * Documentación de la creación y comprobación del *WorkFlow* (flujo de trabajo) para automatizar la construcción de la web estática con MkDocs.

* **[Despliegue con Docker y Nginx](docker.md)**
  * Describe las operaciones de creación y puesta en marcha del servicio **NGinx** en un contenedor Docker para mostrar la documentación de la actividad.

* **[Conclusiones](conclusiones.md)**
  * Apartado final con las conclusiones personales sobre el desarrollo de la actividad.
